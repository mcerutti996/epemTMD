# epemCrossSection

This code computes the electron-positron cross section.

## Dependencies

The following external libraries are required:

- [``LHAPDF6``](https://lhapdf.hepforge.org),
- [``APFEL++``](https://github.com/vbertone/apfelxx)

Make sure they are installed and properly configured in your environment.

## Installation 

The code can be compiled using the following procedure:

```Shell
cd epemTMD
make
```
